## Problem Statement
Analyze time-based trends in stock price data.

## Dataset
- Source: Generated stock price data
- Total Trading Days: 1006
- Period: 2020-2024

## Approach
- Loaded stock price data
- Plotted price trend over time

- Analyzed yearly trends
- Analyzed volume over time

## Key Insights

- Lowest Price: $54.16

- Stock shows upward trend over time
- Volume varies significantly
